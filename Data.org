funcion OrdenamientoPorRanking (D, P):
    // Input
    for each processor P_i in parallel:
        D_i = D[local data of P_i]

    // Gossip
    for each processor P_i in parallel:
        for each processor P_j:
            Send D_i to P_j
        Receive D_j from all processors

    // Broadcast
    for each processor P_i in parallel:
        D_total = concatenate(D_1, D_2, ..., D_P)

    // Sort
    for each processor P_i in parallel:
        Sort(D_total)

    // Local ranking
    for each processor P_i in parallel:
        Local_Rank(D_total)

    // Reduce
    for each processor P_i in parallel:
        Ajuste_global (D_total)

    return D_total
